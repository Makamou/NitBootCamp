# Objective

---

# Key Concepts

## 📦 Repository (Repo)

A collection of files and folders used to store a project.

---

## 💻 Local Repository

A repository stored on your computer.

---

## 🌐 Remote Repository

A repository stored online (GitHub).

---

## 🔑 Difference

| Local Repo | Remote Repo |
| ---------- | ----------- |
| Your PC    | GitHub      |
| Offline    | Online      |

---
