# Linux Virtual Machine (VM)

A Virtual Machine is a software-based computer that allows you to run Linux without requiring a separate physical machine.

### Key Concepts

* VM = Virtual Computer
* Console = Direct access to the Linux system
* Login using provided credentials
* Linux administration is primarily performed through the command line

![Linux Virtual Machine](images/linux-vm.png)

# Understanding the Linux Shell

The **Shell** is the interface between the user and the operating system.

### Components

| Component   | Purpose               |
| ----------- | --------------------- |
| Shell       | User interface        |
| Interpreter | Translates commands   |
| Kernel      | Executes instructions |

### Workflow

```text
User → Shell → Kernel → Output
```

![Linux Shell Architecture](images/linux-shell.png)

# Essential Linux Commands

These are your first commands and should be memorized.

| Command       | Description                |
| ------------- | -------------------------- |
| `whoami`      | Show current user          |
| `pwd`         | Show current directory     |
| `ls`          | List files and folders     |
| `date`        | Display date and time      |
| `clear`       | Clear terminal screen      |
| `ip a`        | Show network configuration |
| `hostnamectl` | Show system information    |
| `uptime`      | Show system uptime         |
| `passwd`      | Change password            |
| `exit`        | Exit shell                 |

### Practice

```bash
whoami
pwd
ls
date
ip a
hostnamectl
uptime
```
